# Server Performance Stats

A simple script designed to collect and display Linux server statistics such as CPU usage, memory utilization, disk space, and other vital system metrics.

Project URL: https://roadmap.sh/projects/server-stats

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/stefanomuraro/devops-p01-server-performance-stats.git
   ```

2. **Grant execution permissions:**
   ```bash
   chmod +x server-stats.sh
   ```

## Usage

Run the script directly from your terminal:
```bash
./server-stats.sh
```
